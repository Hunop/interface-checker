#interface-checker
###Backend to check device for :
<ol>
<li>Interface list</li>
<li>Administrative status</li>
<li>Operational status</li>
</ol>

###Requirements:
<ol>
<li>Python2</li>
<li>pysnmp</li>
<li>PrettyTable - If using direct from CLI</li>
</ol>


###Example usage:
    Darren-MBP:interface-checker darren$ ./1.py pehdrt99
